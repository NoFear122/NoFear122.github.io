ООП - методология программирования в которой весь код нашей программы представляется в виде классов и объектов. Программа = какие-то классы + какие-то объекты. Методология в программировании - способ организации кода программы. Класс - это тип данных который описывает некую сущность. Объект - отдельный представитель класса имеющий поведение и состояние определённое классом. Экземпляр класса = объект. На основе одного класса можно создавать сколь угодное кол-о объектов
